# TicTacToe Game

I plan for this to be a multiplayer TicTacToe game with elo points, accounts, chat system, friend requests. 


---
Here are the stages of development, with a checkmark emoji representing where I am currently at:
1. CLI game of TicTacToe ✅
2. TicTacToe with a in browser front end.
3. Multiplayer TicTacToe
4. TicTacToe, with accounts, elo, and chats.

---
This is purely a project built for fun and learning experience, and will probably not serve a greater purpose than that.
